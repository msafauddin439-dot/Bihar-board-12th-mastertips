# Next Education - Project Overview

## Description
A learning platform specifically designed for Bihar Board Class 10 & 12 students.

## Subjects
- Physics
- Chemistry
- Biology
- Mathematics
- Hindi
- English

## Technical Context
The user provided React Native / Expo code snippets for a mobile app structure:
- Navigation: Native Stack
- Screens: Splash, Login, Signup, Home, Dashboard
- Content: Subject lists and basic branding.

## Goal
Translate this mobile app concept into a high-fidelity web platform for desktop.
